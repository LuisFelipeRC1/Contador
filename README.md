# Corporate AI Agent

Projeto profissional de agente de IA com LLM, RAG, APIs externas e arquitetura inspirada em sistemas corporativos.

## 🚀 Funcionalidades
- Agente com tomada de decisão (ReAct)
- Integração com API de CEP
- Integração com API de clima
- Pipeline simples de RAG
- Uso de embeddings
- Estrutura modular (agent, tools, rag, services)

## 🧠 Conceitos aplicados
- LLM (OpenAI)
- Prompt Engineering
- Reasoning (ReAct)
- APIs REST
- RAG (Retrieval-Augmented Generation)

## ▶️ Como rodar
```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```

## 🎯 Objetivo
Projeto de portfólio para demonstrar desenvolvimento de agentes de IA corporativos.
